## 기능 동작 확인 사항

- **실시간 필터링**: 입력값 변경 즉시 API 재조회 (별도 검색 버튼 불필요)
- **필터 초기화**: "필터 초기화" 버튼 클릭 시 전체 목록 복원 정상 동작
- **총 건수 표시**: 우측 상단에 현재 조회 건수 실시간 반영
- **카테고리 배지**: 카테고리별 색상 코드 정상 표시 (식료품-인디고, 외식-오렌지, 교통-틸, 문화-퍼플)
- **Empty State**: 조건에 맞는 데이터 없을 때 안내 메시지 표시
